Hence it will give the ability to validate a few	network tests: routing of the local and remote IPs, firewall of SMTP transactions from the	server it is run on, outbound NAT of the local IPs on port 25.Usage:	1. Unzip file in a local directory on the server.	2. If neccessary edit the config.php file (only if specific tests should be made).	3. Launch a cmd console and change dir to the directory where files have been unzipped.	4. Launch the command "php nat_validate.php".	5. The result of the NAT will display.
